# 游戏玩法设计文档

## 第一关：神秘的保健室

### 通关流程：
1. 初始场景（保健室）
   - 检查药柜 → 获得线索：药品丢失
   - 检查地面 → 获得线索：猫毛
   - 检查药品清单 → 获得线索：药品记录

2. 与张老师对话
   - 询问小红去向
   - 展示药品丢失线索 → 获得新对话选项
   - 展示猫毛线索 → 获得线索：猫的信息

3. 前往走廊
   - 检查公告栏 → 获得线索：值日表
   - 与王大爷对话
   - 展示猫毛和药品记录 → 获得线索：生病的猫

4. 线索组合规则：
   - 药品丢失 + 猫毛 = 可疑的痕迹
   - 药品记录 + 值日表 = 小红的行动
   - 猫的信息 + 生病的猫 = 流浪猫的下落
   - 可疑的痕迹 + 流浪猫的下落 = 后院位置

5. 解锁后院
   - 需要收集：后院位置线索
   - 发现小红和生病的猫
   - 完成第一关

### 线索组合详细规则：
1. 可疑的痕迹
   - 组合条件：药品丢失 + 猫毛
   - 新线索描述：药品丢失和猫毛的发现暗示有人带着药品接触过猫
   - 解锁：与张老师的新对话选项

2. 小红的行动
   - 组合条件：药品记录 + 值日表
   - 新线索描述：小红在值日时间取走了感冒药
   - 解锁：与王大爷的新对话选项

3. 流浪猫的下落
   - 组合条件：猫的信息 + 生病的猫
   - 新线索描述：后院有一只生病的流浪猫
   - 解锁：后院场景的线索

4. 后院位置
   - 组合条件：可疑的痕迹 + 流浪猫的下落
   - 新线索描述：小红可能带着药去后院照顾生病的猫
   - 解锁：后院场景

### 关键NPC对话条件：
1. 张老师
   - 初始对话：无条件
   - 询问药品：需要药品丢失线索
   - 询问猫毛：需要猫毛线索

2. 王大爷
   - 初始对话：无条件
   - 询问后院：需要猫的信息线索
   - 询问照顾猫的人：需要猫毛和药品记录线索

### 场景解锁条件：
1. 走廊
   - 初始解锁

2. 后院
   - 需要获得"后院位置"线索
   - 或完成特定对话分支

### 结局条件：
- 找到小红和生病的猫
- 理解小红的行为动机
- 获得完整的故事线索
